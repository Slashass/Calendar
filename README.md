This folder is a copy of the project renamed to `trading-calendar`.

Favicons:
- The project includes `assets/icon.svg` (the vector icon).
- To generate `assets/favicon.png` and `assets/favicon.ico` run:

```bash
chmod +x scripts/generate-favicons.sh
./scripts/generate-favicons.sh
```

This requires ImageMagick (`convert` or `magick`). On macOS:

```bash
brew install imagemagick
```

After generating, open `index.html` in this folder.
